# AI Token Saver - Multi Support

この拡張機能は、AIへの送信前に「トークン量」を可視化し、ユーザーの節約意識を高めるためのツールです。

## 概要
AIモデル（ChatGPT, Claude, Gemini等）に送信するテキストの量をリアルタイムで計算します。
「自分がどれだけの情報を送ろうとしているか」を把握することで、冗長な表現を削り、効率的なプロンプト作成を支援します。

## 対応プラットフォーム
- **ChatGPT** (`chatgpt.com`)
- **Claude** (`claude.ai`)
- **Gemini** (`gemini.google.com`)
- **ExaBase** (`*.exabase.ai`)

## インストール手順
1. このフォルダをローカルの任意の場所に保存します。
2. Google Chrome (またはEdge) を開き、URLバーに `chrome://extensions/` を入力します。
3. 右上の **「デベロッパー モード」** をオンにします。
4. **「パッケージ化されていない拡張機能を読み込む」** をクリックします。
5. このフォルダを選択して読み込みます。

## 特徴
- **リアルタイム表示**: 入力中、即座に推定トークン数を算出。
- **アラートカラー**: 
  - 800トークン未満: **緑** (効率的)
  - 800〜2000トークン: **オレンジ** (見直し推奨)
  - 2000トークン以上: **赤** (要圧縮)
- **プライバシー配慮**: AIとの通信は行わず、ブラウザ内のみで計算を実行します。

## 節約のヒント
- 挨拶や定型句を削る。
- 「です・ます」を「だ・である」または体言止めにする。
- AIに対して「簡潔に答えて」と指示を添え、出力トークンも節約する。
