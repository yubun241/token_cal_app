# AI Token Saver Pro (Robust Edition)

AIチャット（ChatGPT, Claude, Gemini, ExaBase）への送信前にトークン量を可視化し、ワンクリックで「AIに最適化された最小限の表現」に圧縮するためのブラウザ拡張機能です。

## 🌟 主な機能

1. **リアルタイム・トークン計測**
   - 入力中のテキストからトークン量を近似計算し、画面右上に常時表示します。
   - トークン量に応じて色が変わります（安全: 緑 / 注意: 黄 / 警告: 赤）。

2. **ワンクリック節約（Click to Save）**
   - **バッジをクリック**するだけで、以下の節約処理を瞬時に実行します。
     - **英数半角化**: トークン効率の悪い全角の１２３やＡＢＣを、半角の123やABCに正規化します。
     - **挨拶・冗長表現のカット**: 「お疲れ様です」「よろしくお願いします」等の定型句を削除します。
     - **文末の圧縮**: 「〜だと思います」→「〜だ」などの簡潔な表現へ変換します。
     - **空白・改行の集約**: 無駄な空行や連続するスペースを最小限に詰め、構造を最適化します。

3. **マルチプラットフォーム対応**
   - 主要なAIサービス（ChatGPT, Claude, Gemini, ExaBase）で動作するよう、高度な入力欄検知ロジックを搭載しています。

## 🚀 インストール方法

1. 本フォルダをローカル環境に保存します。
2. Google Chrome または Edge を開き、拡張機能ページ（ `chrome://extensions/` ）にアクセスします。
3. 右上の **「デベロッパー モード」** をオンにします。
4. **「パッケージ化されていない拡張機能を読み込む」** をクリックし、このフォルダを選択します。

## 💡 節約のポイント

* **バッジをクリックしてから送信**: 人間にとっての「丁寧さ」を、ボタン一つでAIにとっての「低コスト・高効率」に変換します。
* **視覚的な意識付け**: カウンターが赤色になったら、文章を分割するか、不要な情報を削る目安になります。
* **Outputの節約**: 本ツールで入力を短くした上で、AIに「簡潔に答えて」と指示することで、返答（Output）にかかるトークンもさらに節約できます。
