# AI Token Counter for ChatGPT

ChatGPTの入力欄に入力されたテキストのトークン量をリアルタイムで推定・表示するブラウザ拡張機能です。

## 機能
- **リアルタイムカウント**: 入力中のテキストからトークン量を近似計算します。
- **視覚的フィードバック**: トークン量が多くなるとバッジの色が変化します（1000トークン超で赤色）。

## インストール方法 (Chrome/Edge)
1. このリポジトリ（またはフォルダ）をローカルに保存します。
2. ブラウザで `chrome://extensions/` を開きます。
3. 右上の **「デベロッパー モード」** をオンにします。
4. **「パッケージ化されていない拡張機能を読み込む」** ボタンをクリックします。
5. この `README.md` が入っているフォルダを選択します。

## 使い方
1. [ChatGPT](https://chatgpt.com/) を開きます。
2. 下部のメッセージ入力欄に文字を入力します。
3. 画面右下に現在の推定トークン数が表示されます。

## 注意事項
- 本機能はトークナイザーのアルゴリズムに基づいた**推定値**を表示します。正確な消費量はAIモデル側の計算に依存します。
- 返答（Output）のトークン量は含まれません。送信前の節約の目安としてご利用ください。
